# Flashscore Workflow

1. Open flashscore.com
2. Handle cookie consent
3. Navigate to basketball
4. Filter by scheduled matches
5. Extract match data
6. Navigate to each match
7. Extract data from SUMMARY, H2H, ODDS, STATS tabs
8. Extract data from tertiary tabs (Inc OT, FT, Q1)
9. Save data to database
10. Repeat for all matches
