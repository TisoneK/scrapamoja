# LLM-Only Workflow Template

A comprehensive template for workflows that rely entirely on LLM guidance and human interaction.

## Purpose

This template provides structure for workflows that require AI decision-making, creative problem-solving, and human collaboration without automation scripts.

## Key Features

- **Pure LLM Integration**: Structured prompts and decision frameworks
- **Human Collaboration**: Clear interaction points and review processes
- **Flexible Processes**: Adaptive workflows that can handle complexity
- **Quality Assurance**: Built-in validation and iteration mechanisms
- **Knowledge Management**: Documentation and learning systems

## Quick Start

1. **Copy Template**: 
   ```bash
   cp -r docs/workflows/workflow-templates/llm-only-template docs/workflows/your-workflow
   ```

2. **Rename Files**: Replace `llm-only-template` with your workflow name

3. **Customize Content**: 
   - Update descriptions and purposes
   - Modify workflow steps
   - Configure LLM rules
   - Design prompt templates

4. **Test**: Validate with sample scenarios

5. **Deploy**: Add to workflow registry

## Template Components

### Core Files
- **start.md**: Entry point with mode selection
- **llm-only-template.md**: Main workflow documentation
- **README.md**: User-friendly overview
- **rules.md**: LLM behavior guidelines
- **status.json**: Progress tracking

### Resources
- **templates/**: Reusable prompt templates
- **examples/**: Sample inputs and outputs
