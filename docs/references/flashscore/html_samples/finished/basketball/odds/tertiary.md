## URL Structure

```
/match/basketball/{home-team-slug}-{home-id}/{away-team-slug}-{away-id}/odds/{market-type}/{time-period}/?mid={match-id}
```

**Common Time Periods:**
- `full-time` - Full time (regulation)
- `ft-including-ot` - Full time including overtime
- `1st-half` - First half
- `1st-qrt` - First quarter

## Notes

### General
- Tertiary tabs filter odds by time period
- Numbers in parentheses indicate available odds count
- Basketball-specific: includes quarter and half options

### Match State Differences
- **Finished:** Shows archived/closing odds with counts
- **Scheduled:** All time periods typically available
- **Live:** Only current/relevant time periods available
